# Nupur Ghangarekar | Portfolio

A premium, cinematic personal portfolio built with React, Vite, and Framer Motion. This project showcases a grounded, articulate, and technically confident digital identity focused on medical AI research and software engineering.

## 🌌 Aesthetic & Narrative
The portfolio is designed with a **luxury minimalism** aesthetic, combining warm charcoal backgrounds, lavender depth glows, and cinematic motion design. The narrative focus is on **Software Engineering & Medical Vision Research**, particularly in high-dimensional CT volume processing.

## 🛠️ Tech Stack
- **Core**: React + Vite
- **Motion**: Framer Motion (Scroll choreography, Magnetic interactions, 3D Tilt)
- **Styling**: Vanilla CSS (Custom Design System)
- **Icons**: Material Icons / Custom SVG

## 📁 Key Sections
- **Selected Work**: High-impact projects including Helios (Medical AI), Attention OS (Automation), and Opsium ML (Predictive Analytics).
- **Professional Experience**: Research roles at VESIT and Software Development at Zemo.
- **Technical Foundations**: Expertise in Computer Vision, Engineering & Cloud, and Forensic Security.
- **Beyond Technology**: Perspectives on debate, philosophy, and technical storytelling.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/NupurGhangarekar/Portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
